# Modelo de negocio - Hidrofix

Esta carpeta contiene el análisis financiero y modelo de negocio para el generador de hidrógeno doméstico.

## Contenido planificado

- `cost_analysis.md` — análisis de costos de componentes y fabricación.
- `pricing.md` — estrategia de precios (USD 85 estimado).
- `market.md` — mercado objetivo: cocinas domésticas, hogares off-grid.
- `roi.md` — retorno de inversión para el usuario final.
