# Simulaciones - Hidrofix

Esta carpeta contiene los modelos de simulación para validar el comportamiento del generador.

## Contenido planificado

- `gas_flow.sim` — simulación de flujo de hidrógeno (ANSYS Fluent).
- `thermal_analysis.sim` — simulación térmica de la celda (temperatura >50°C).
- `backup_power.sim` — simulación del supercondensador durante corte de energía.

## Parámetros clave

- Flujo de hidrógeno: 0–10 L/min.
- Temperatura de operación: <45°C (normal), >50°C (alerta).
- Respaldo energético: 5F, 5.5V, autonomía 5–10 segundos.
